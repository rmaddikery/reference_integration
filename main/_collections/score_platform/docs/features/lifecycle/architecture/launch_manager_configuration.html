Launch Manager Configuration#

The Launch Manager supports a set of configuration parameters, which are grouped in the following categories:

Component Configuration Parameters#

These parameters specify the arguments needed for spawning a process.

  • Process user, group IDs sup... (feat_req__lifecycle__uid_gid_support)

  • Supplementary group support (feat_req__lifecycle__supplementary_groups)

  • CPU runmask support (feat_req__lifecycle__runmask_support)

  • Process priority support (feat_req__lifecycle__launch_priority_support)

  • CWD support (feat_req__lifecycle__cwd_support)

  • Handling process args (feat_req__lifecycle__process_launch_args)

  • Standard handle redirection (feat_req__lifecycle__std_handle_redir)

  • ASLR support (feat_req__lifecycle__aslr_support)

  • Resource limit support (feat_req__lifecycle__process_rlimit_support)

  • Security policy support (feat_req__lifecycle__support_secpol_type)

Dependency parameters#

These parameters specify how the components depend on each other.

Recovery parameters#

These parameters specify the recovery actions for a component, when it fails.

Alive monitoring parameters#

These parameters specify the alive monitoring rules for the application.

Requirements related to the external monitoring#

These parameters specify how the Launch Manager itself is monitored.

  • Launch manager self health ... (feat_req__lifecycle__lm_self_health_check)

  • Launch manager external wat... (feat_req__lifecycle__lm_ext_watchdog_notify)

  • Launch manager external wat... (feat_req__lifecycle__lm_ext_wdg_failed_test)

  • Launch manager external mon... (feat_req__lifecycle__lm_ext_watchdog_cfg)
