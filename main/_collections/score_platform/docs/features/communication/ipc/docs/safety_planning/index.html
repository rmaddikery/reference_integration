Tailoring#

Additional to the tailoring in the SW platform project as defined in the Platform Safety Plan (doc__platform_safety_plan) we define here the additional tailoring on feature level.

  • Excluded for this feature are additionally the following work products (and their related requirements):

    • Requirements Inspection (wp__requirements_inspect): in version 0.5 the inspections have to be done by the user

    • Architecture Verification (wp__sw_arch_verification): in version 0.5 the inspections have to be done by the user
