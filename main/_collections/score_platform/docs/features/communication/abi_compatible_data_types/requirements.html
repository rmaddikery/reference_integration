Vector#

Provide AbiVec<T,N>
status: valid
security: NO
safety: QM
reqtype: Functional
satisfies: stkh_req__communication__abi_compatible

An ABI-compatible AbiVec<T, N> type shall be provided in both C++ and Rust with the specified layout.

#[repr(C)]
pub struct AbiVec<T, const N: usize> {
   len: u32,
   elements: [T; N],
}

template<typename T, std::uint32_t N>
struct AbiVec {
private:
   std::uint32_t len;
   T elements[N];
};

Result#

Provide AbiResult<T,E>
status: valid
security: NO
safety: QM
reqtype: Functional
satisfies: stkh_req__communication__abi_compatible

An ABI-compatible AbiResult<T, E> type shall be provided in both C++ and Rust with the specified layout.

#[repr(C)]
pub struct AbiResult<T, E> {
   is_err: bool,
   value: AbiResultUnion<T, E>,
}

#[repr(C)]
union AbiResultUnion<T, E> {
   ok: ManuallyDrop<T>,
   err: ManuallyDrop<E>,
}

template<typename T, typename E>
struct AbiResult {
private:
   AbiBool is_err;
   union {
      T ok;
      E err;
   } value;
};
